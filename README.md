# Proyecto Final - MDEX

Repositorio con los archivos del proyecto final del curso MDEX.

- Archivos principales:
  - `app.py` : aplicación principal (Python).
  - `datos_aguacate_masivos.csv` : conjunto de datos (si lo subes al repo, ten en cuenta su tamaño).
  - `proyecto_agronomía.html` : documento de salida / informe.

Instrucciones rápidas:

1. Crear un repo en GitHub (o usar `gh` CLI) y copiar la URL remota.
2. En la carpeta del proyecto ejecutar (PowerShell):

```
git remote add origin <URL_DEL_REPO>
git push -u origin main
```
